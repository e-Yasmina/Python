Python Basics: A Step-by-Step Guide

Step 1: Variables and Data Types

Variables are containers for storing data values. In Python, you don't need to declare the type of a variable; it's determined automatically.

Common Data Types:

  • String (str): Text, enclosed in quotes. Example: "Hello"
  • Integer (int): Whole numbers. Example: 10
  • Float (float): Decimal numbers. Example: 3.14
  • Boolean (bool): Represents True or False.
# String
name = "Alice"

# Integer
age = 30

# Float
height = 5.5

# Boolean
is_student = True

print(name)
print(age)

Step 2: Flow Control (if/else)

Flow control allows your program to make decisions. The if, elif (else if), and else statements are used to execute different blocks of code based on conditions.

age = 18

if age < 13:
    print("You are a child.")
elif age < 20:
    print("You are a teenager.")
else:
    print("You are an adult.")

Step 3: Lists

A list is a collection of items which is ordered and changeable. Lists are written with square brackets.

You can add items using .append() and access them by their index (starting from 0).

# Create a list
fruits = ["apple", "banana", "cherry"]
print(fruits)

# Access an item
print(fruits[1])  # Output: banana

# Add an item
fruits.append("orange")
print(fruits)

Step 4: Loops

Loops are used to execute a block of code repeatedly.

for loop: Used for iterating over a sequence (like a list or a range).

# Loop through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Loop a specific number of times
for i in range(5):  # from 0 to 4
    print(f"Loop number {i}")
