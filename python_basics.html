Python Basics: A Step-by-Step Guide

Step 1: Variables and Data Types

Think of a variable as a labeled box where you can store information. You give the box a name (the variable name) and put something inside it (the value).

In Python, you create a variable by giving it a name and using the equals sign (=) to assign it a value. Python is smart enough to figure out the type of data you're storing.

Naming Rules

  • A variable name must start with a letter or the underscore character (_).
  • A variable name cannot start with a number.
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ ).
  • Variable names are case-sensitive (age, Age and AGE are three different variables).

Common Data Types

  • String (str): Used for text. You must enclose them in single (') or double (") quotes.
  • Integer (int): Used for whole numbers, without decimals.
  • Float (float): Used for numbers with decimals.
  • Boolean (bool): A special type that can only be True or False.
# String: for storing text
greeting = "Hello, Python learner!"
# Integer: for whole numbers
user_age = 25
# Float: for decimal numbers
pi_value = 3.14159
# Boolean: for true/false values
is_learning = True

print(f"User's age: {user_age}")

Step 2: Working with Strings

Strings are one of the most common data types. Since they are sequences of characters, you can do a lot more than just store text.

String Concatenation

You can combine strings using the + operator.

first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name) # Output: John Doe

Indexing and Slicing

You can access parts of a string using indexes (positions) in square brackets []. Slicing lets you get a substring.

sentence = "The quick brown fox"
first_char = sentence[0] # 'T' (index 0)
word = sentence[4:9] # 'quick' (from index 4 up to, but not including, 9)
print(f"The first character is: {first_char}")
print(f"A sliced word is: {word}")

Common String Methods

Methods are special functions that belong to an object. Here are a few useful ones for strings:

  • .upper(): Converts the string to uppercase.
  • .lower(): Converts the string to lowercase.
  • .strip(): Removes any leading and trailing whitespace.
  • len(): A built-in function to get the length of the string.
messy_text = "   Some Text Here   "
print(f"Original: '{messy_text}'")
print(f"Stripped: '{messy_text.strip()}'")
print(f"Uppercase: '{messy_text.upper()}'")
print(f"Length of stripped text: {len(messy_text.strip())}")

Challenge: String Cleanup

Step 3: Flow Control (if/else)

Flow control lets your code make decisions. It's like a fork in the road.

You can also combine conditions with logical operators:

  • and: Both conditions must be true.
  • or: At least one condition must be true.
  • not: Inverts the result (True becomes False and vice-versa).
age = 22
has_license = True

if age >= 18 and has_license:
    print("You are allowed to drive.")
elif age >= 18 and not has_license:
    print("You are old enough, but you need a license to drive.")
else:
    print("You are not allowed to drive.")

Step 4: Lists

A list is an ordered, changeable collection of items. They are one of the most versatile data types in Python.

Modifying a List

You can change, add, and remove items.

  • Change an item by accessing its index: my_list[1] = "new_item".
  • Add an item with .append().
  • Remove an item with .remove("item_name") or del my_list[index].
# A list of tasks
tasks = ["wash car", "buy groceries", "pay bills"]
print(f"Original tasks: {tasks}")

# Change an item
tasks[0] = "wash the dog"
print(f"Updated tasks: {tasks}")

# Remove an item
tasks.remove("pay bills")
print(f"Final tasks: {tasks}")

Step 5: Loops

Loops are for doing something over and over. This is known as "iteration".

The for loop with range()

The range() function is flexible. You can give it a start, stop, and step: range(start, stop, step).

# Countdown from 5 down to 1
for i in range(5, 0, -1):
    print(f"Countdown: {i}")
print("Blast off!")

The while loop

A while loop is perfect for when you want to repeat something until a certain condition is met.

# Keep asking for input until the user types 'quit'
command = ""
while command.lower() != "quit":
    command = input("Enter a command (or 'quit' to exit): ")
    print(f"You entered: {command}")

print("Exiting the program.")

Challenge: Sum of Numbers

Use a for loop and range() to calculate the sum of all numbers from 1 to 100.

total = 0
# range(1, 101) goes from 1 up to, but not including, 101.
for number in range(1, 101):
    total = total + number

print(f"The sum of numbers from 1 to 100 is: {total}")
