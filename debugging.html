Writing code is great, but a big part of programming is finding and fixing errors, a process called debugging. Let's look at the three main types of errors you'll encounter.

1. Syntax Errors

These are like grammar mistakes in a sentence. You've broken a rule of the Python language, and the program can't even start to run. These are often the easiest to fix.

Example of a Syntax Error: A missing colon : at the end of a loop definition. 
for i in range(5)  # Missing a colon here!
    print(i)
When you try to run this, you'll get an error message that looks something like this, pointing right at the problem: 
  File "my_script.py", line 1
    for i in range(5)
                     ^
SyntaxError: invalid syntax
Example of an IndentationError: Incorrect indentation breaks Python's structure. 
def greet_user():
print("Hello!")  # This line is not indented properly!
This gives an IndentationError because Python expects the function body to be indented. 
# The corrected way - proper indentation
def greet_user():
    print("Hello!")  # Now it's properly indented!

greet_user()
Another Syntax Error Example: Mismatched quotes in a string. 
message = "Hello, world!'  # Opening with double quote but closing with single!
You'll get a SyntaxError because the quotes don't match. 
# The corrected way - matching quotes
message = "Hello, world!"  # Both opening and closing with double quotes
# OR
message = 'Hello, world!'  # Both opening and closing with single quotes
print(message)

Challenge: Fix the Syntax

The following code has a syntax error. Can you find and fix it?

def say_hello(name)
    print(f"Hello, {name}")

say_hello("World")

Answer: The function definition was missing a colon :.

def say_hello(name):  # Added the colon
    print(f"Hello, {name}")

say_hello("World")

2. Runtime Errors (Exceptions)

These errors happen while your program is running. The syntax is correct, but the code tries to do something impossible. These are also called "Exceptions".

Example of a TypeError: Trying to add a number and a string. 
age = 25
print("Your age is: " + age) # Can't add a string and an integer
The program crashes and gives you a TypeError. The fix is to convert the number to a string first, or use an f-string. 
# The corrected way with an f-string
age = 25
print(f"Your age is: {age}")

# OR convert the number to a string
print("Your age is: " + str(age))

# OR use the format method
print("Your age is: {}".format(age))
Example of a NameError: Using a variable that hasn't been created yet. 
print(my_variable) # my_variable was never defined!
This gives a NameError because Python doesn't know what my_variable is.
Another example of a NameError: Using a function that doesn't exist. 
result = non_existent_function(5) # This function doesn't exist!
This also gives a NameError because Python doesn't know what non_existent_function is. 
# The corrected way - define the function or use correct function name
def multiply_by_two(x):  # Define the function
    return x * 2

result = multiply_by_two(5)  # Now this will work
print(result)  # This will print 10

# Or use an existing function
import math
result = math.sqrt(25)  # Using the correct function name
print(result)  # This will print 5.0
Example of an IndexError: Trying to access an index that doesn't exist in a list. 
fruits = ['apple', 'banana', 'orange']
print(fruits[5]) # Index 5 doesn't exist in the list!
This gives an IndexError because Python is trying to access an element at position 5, but the list only has indices 0, 1, and 2. 
# The corrected way - check the length first or use a valid index
fruits = ['apple', 'banana', 'orange']
if len(fruits) > 5:
    print(fruits[5])
else:
    print(f"The list only has {len(fruits)} fruits, so index 5 doesn't exist.")

# Or access a valid index
print(fruits[1]) # This will print 'banana'
Example of a ValueError: Passing an argument of the correct type but with an inappropriate value. 
number = int("hello") # Can't convert "hello" to an integer!
This gives a ValueError because Python can't convert the string "hello" into an integer. 
# The corrected way - validate the input first
text = "hello"
if text.isdigit():
    number = int(text)
else:
    print(f"Cannot convert '{text}' to an integer because it's not a valid number.")

# Another example: converting a valid number string
valid_number = int("123") # This works fine
print(valid_number) # This will print 123
Example of a ZeroDivisionError: Dividing a number by zero. 
numerator = 10
denominator = 0
result = numerator / denominator # You can't divide by zero!
This gives a ZeroDivisionError because division by zero is mathematically undefined. 
# The corrected way - check if denominator is zero first
numerator = 10
denominator = 0
if denominator != 0:
    result = numerator / denominator
    print(f"The result is: {result}")
else:
    print("Error: Cannot divide by zero!")

# Another example with a valid calculation
result = numerator / 2  # This works fine
print(result) # This will print 5.0
Example of an AttributeError: Trying to access an attribute or method that doesn't exist for an object. 
text = "Hello, World!"
uppercase_text = text.upppercase() # Typo in the method name!
This gives an AttributeError because the string object doesn't have a method named 'upppercase'. The correct method name is 'upper'. 
# The corrected way - using the correct method name
text = "Hello, World!"
uppercase_text = text.upper()  # The correct method name is 'upper'
print(uppercase_text)  # This will print 'HELLO, WORLD!'

Challenge: Fix the Runtime Error

This code tries to calculate the total price but fails. Find and fix the error.

item_price = "50" # Price is stored as a string
quantity = 3
total_price = item_price * quantity

print(f"Total price: {total_price}")

Answer: You can't multiply a string ("50") by a number. You need to convert the price to an integer first using int().

item_price = "50"
quantity = 3
# Convert item_price to an integer before multiplying
total_price = int(item_price) * quantity

print(f"Total price: {total_price}") # Output: Total price: 150

3. Logic Errors

These are the trickiest bugs. The code runs without crashing, but it produces the wrong result. The computer is doing exactly what you told it to do, but what you told it to do was wrong.

Example of a Logic Error: Calculating the wrong average. 
numbers = [1, 2, 3, 4, 5]
total = sum(numbers)
# Oops! There are 5 numbers, but we divided by 4.
average = total / 4
print(f"The average is: {average}") # Prints 3.75 instead of 3.0
The code runs, but the result is incorrect. The only way to find this is by testing your code and realizing the output is not what you expect.
Another Example of a Logic Error: Off-by-one error in a loop. 
for i in range(1, 10):  # This will print 1 to 9, not 1 to 10!
    print(i)

# If you want to include 10, you need range(1, 11)
The code runs without errors, but it doesn't produce the expected result. You wanted numbers 1 to 10, but got only 1 to 9. 
# The corrected way - to include 10 in the range
for i in range(1, 11):  # Now this will print 1 to 10
    print(i)
# OR if you want 10 numbers starting from 1, use range(1, 11)
numbers_list = list(range(1, 11))
print("Numbers:", numbers_list)

A common way to find logic errors is to use print() statements to check the value of your variables at different points in your code to see where things go wrong.

Challenge: Fix the Logic

This function is supposed to return a list of all numbers greater than 10, but it's not working correctly. Can you fix the logic?

def find_large_numbers(numbers):
    large_nums = []
    for num in numbers:
        if num < 10: # This condition is wrong!
            large_nums.append(num)
    return large_nums

my_numbers = [5, 12, 3, 15, 8, 20]
result = find_large_numbers(my_numbers)
print(f"The large numbers are: {result}") # Expected: [12, 15, 20]

Answer: The condition in the if statement was backwards. It should be checking for numbers > 10, not < 10.

def find_large_numbers(numbers):
    large_nums = []
    for num in numbers:
        if num > 10: # Corrected the condition
            large_nums.append(num)
    return large_nums

my_numbers = [5, 12, 3, 15, 8, 20]
result = find_large_numbers(my_numbers)
print(f"The large numbers are: {result}")
