Game 1: Number Guessing Game

Goal: Write a program where the computer thinks of a number and the player has to guess it.

Concepts: import, random.randint(), while loops, if/elif/else, input(), and try/except for error handling.

Fill in the code below. The game should give hints if the guess is too high or too low.

import random

secret_number = random.randint(1, 50)
guess = 0
attempts = 0

print("I'm thinking of a number between 1 and 50.")

while guess != secret_number and attempts < 5:
    try:
        guess = int(input("Take a guess: "))
        attempts += 1
        if guess < secret_number:
            print("Too low!")
        elif guess > secret_number:
            print("Too high!")
    except ValueError:
        print("Please enter a number.")

if guess == secret_number:
    print(f"Good job! You guessed it in {attempts} attempts!")
else:
    print(f"Sorry, the number was {secret_number}.")

Hint: The while loop should continue as long as the guess is not correct and the number of attempts is within the limit.

Game 2: Hangman

Goal: Create the classic word-guessing game, Hangman.

Concepts: Lists, strings, for loops, and checking for items in a list.

The computer will pick a secret word. The player guesses letters to figure it out. Fill in the logic.

import random
word_list = ["python", "gemini", "code"]
secret_word = random.choice(word_list)
display = ["_"] * len(secret_word)
game_over = False
lives = 6

print("Welcome to Hangman!")

while not game_over and lives > 0:
    print(f"\nYou have {lives} lives left.")
    print(f"{ ' '.join(display)}")
    
    guess = input("Guess a letter: ").lower()

    if guess in display:
        print(f"You've already guessed '{guess}'.")
        continue

    # Check if the guessed letter is in the word
    for i in range(len(secret_word)):
        if secret_word[i] == guess:
            display[i] = guess

    # If the letter is not in the word, reduce lives
    if guess not in secret_word:
        print(f"'{guess}' is not in the word.")
        lives -= 1
    
    # Check if the player has won
    if "_" not in display:
        game_over = True
        print("\nCongratulations, you win!")
        print(f"The word was: {secret_word}")

# If the loop finishes and game is not over, the player has lost
if not game_over:
    print("\nYou lose!")
    print(f"The word was: {secret_word}")
Game output will appear here.
Hint: Use a list called display to hold the underscores and correctly guessed letters. Use a for loop to update it each turn.

Game 4: Tic-Tac-Toe

Goal: Create a two-player Tic-Tac-Toe game.

Concepts: 2D lists (a list of lists) to represent the board, and functions to manage game logic like checking for a win.

You need to create functions to print the board, check for a winner, and manage player turns.

# --- Functions ---
def print_board(board):
    for row in board:
        print(" | ".join(row))
        print("-" * 9)

def check_win(board, player):
    # Check rows, columns, and diagonals
    for i in range(3):
        if all([board[i][j] == player for j in range(3)]) or \
           all([board[j][i] == player for j in range(3)]):
            return True
    if all([board[i][i] == player for i in range(3)]) or \
       all([board[i][2 - i] == player for i in range(3)]):
        return True
    return False

# --- Game Setup ---
board = [[" " for _ in range(3)] for _ in range(3)]
current_player = "X"
game_is_running = True

# --- Main Game Loop ---
while game_is_running:
    print_board(board)
    try:
        row = int(input(f"Player {current_player}, enter row (0, 1, 2): "))
        col = int(input(f"Player {current_player}, enter col (0, 1, 2): "))

        if board[row][col] == " ":
            board[row][col] = current_player
            if check_win(board, current_player):
                print_board(board)
                print(f"Player {current_player} wins!")
                game_is_running = False
            else:
                # Switch player
                current_player = "O" if current_player == "X" else "X"
        else:
            print("That spot is already taken!")
            
    except (ValueError, IndexError):
        print("Invalid input. Please enter a number between 0 and 2.")
